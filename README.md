# BusArrival
Android 대중교통 길찾기 앱

## 프로젝트 소개

BusArrival는 SK Open API(Tmap)를 사용하여  
출발지와 도착지를 입력하면 대중교통 경로를 검색하고  
소요 시간, 거리, 요금, 환승 정보를 제공하는 Android 애플리케이션입니다.

Jetpack Compose와 MVVM 구조로 구현되었습니다.

## 주요 기능

- 출발지 / 도착지 장소 검색
- 대중교통 경로 검색 (버스, 지하철, 도보)
- 소요 시간, 거리, 요금 표시
- 환승 횟수 표시
- 구간별 이동 정보 제공

## 사용 API

- SK Open API (Tmap)
  - 장소 검색 API
  - 대중교통 경로 검색 API
